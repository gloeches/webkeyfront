Enterprise Filter Update

This document details the changes needed to update the enterprise filter functionality. The goal is to remove the "Filter by name" button and have the list filter automatically as you type in the text box.

File to Modify: src/app/pages/enterprises/enterprises.component.html

The only file that needs to be changed is the HTML template for the enterprises component.

Before


<div class="d-flex justify-content-center p-2 bd-highlight">
    <div class="p-2">
        <label for="Company Name">Company Name: </label>
    </div>
    <div class="p-2">
        <input id="Company Name" #enterpriseName>
    </div>
    <div class="p-1">
        <button class="btn btn-info p-2" type ="button" (click)="getEnterprisesLikeName(enterpriseName.value)">
            <span> Filter by name</span>
        </button>
    </div>
    <div class="p-1">
        <button class="btn btn-info p-2" type ="button" (click)="createEnterprise()">
            <span> Create Enterprise</span>
        </button>
    </div>
   
</div>

After


<div class="d-flex justify-content-center p-2 bd-highlight">
    <div class="p-2">
        <label for="Company Name">Company Name: </label>
    </div>
    <div class="p-2">
        <input id="Company Name" #enterpriseName (input)="getEnterprisesLikeName(enterpriseName.value)">
    </div>
    <div class="p-1">
        <button class="btn btn-info p-2" type ="button" (click)="createEnterprise()">
            <span> Create Enterprise</span>
        </button>
    </div>
   
</div>

No Changes Needed: src/app/pages/enterprises/enterprises.component.ts

No changes are needed in the TypeScript file. The existing getEnterprisesLikeName method is already set up to handle the filtering logic.
